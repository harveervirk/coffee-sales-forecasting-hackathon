Cleaning Methodology

pd.to_datetime(df["Transaction Date"], errors="coerce") was applied to all date values. Any row where the date could not be parsed returned NaT. These 240 rows are counted as invalid and excluded from the forecasting time series, but their revenue is counted in aggregate summaries.

Values of UNKNOWN and ERR_PM_102 were flagged as unknown. These 555 rows are retained because the revenue is real — only the payment channel is unidentifiable.

All Item and Province values had whitespace stripped with .str.strip() and consistent casing applied with .str.title(), preventing duplicates in product and regional aggregations.
