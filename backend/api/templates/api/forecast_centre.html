Seasonal Naive Baseline

Predicts each day's revenue as equal to the same day 7 days earlier. Simple, interpretable, captures weekly patterns.

Holt-Winters Exponential Smoothing

Captures level, trend, and weekly seasonality (seasonal_periods=7). Uses additive components via statsmodels.

Error Metrics Explained
MAE – Mean Absolute Error
Average dollar difference between forecast and actual. Easier to interpret because it's in the same unit as revenue. Used for model selection.
RMSE – Root Mean Squared Error
Penalises large errors more than MAE. Useful for flagging occasional big misses.
MAPE – Mean Absolute Percentage Error
Average error as a percentage of actual. Zero-revenue days are excluded to prevent division errors.
