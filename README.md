Coffee Sales Forecasting Hackathon

This repository contains our three-day hackathon project for analyzing historical coffee sales and forecasting sales for the next six months.

Technology Stack

* Python
* Django
* Power BI
* Forecasting model
* GitHub

Project Structure

* backend/ - Django backend
* analysis/ - Data analysis and forecasting work
* dashboard/ - Power BI dashboard files
* data/ - Project datasets
* docs/ - Documentation
* presentation/ - Final presentation files

Backend Setup

Create and activate a virtual environment:

python3 -m venv .venv
source .venv/bin/activate

Install dependencies:

python -m pip install -r backend/requirements.txt

Run migrations:

python backend/manage.py migrate

Start the Django development server:

python backend/manage.py runserver

Open the application at:

http://127.0.0.1:8000/

Team Workflow

1. Clone the repository.
2. Create a branch for your task.
3. Make and test your changes.
4. Commit and push your branch.
5. Open a pull request into main.

Do not commit passwords, API keys, .env files, virtual environments, or confidential datasets.
